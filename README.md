# How to get set up

## Things you need

* NVM
* Node
* Yarn
* Webpack
* Babel
* React
* Tape (BlueTape)


## Install Node Version Manager (NVM)
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.4/install.sh | bash
source ~/.bashrc
```

## Install node
```
nvm install --lts
```
